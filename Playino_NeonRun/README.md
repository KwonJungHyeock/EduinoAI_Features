# 🤖 EDDIE NEON RUN — 3D 러너

Three.js 기반 **3D 엔드리스 러너**. EDDIE 로봇이 네온 트랙을 질주하며 장애물을 피하고 코인을 먹는다.
("조작감 나는 진짜 게임" 방향으로 새로 만든 자유주제 버전)

## 실행
`index.html`을 브라우저로 더블클릭 → 바로 실행 ▶
> ⚠️ Three.js를 CDN에서 불러오므로 **인터넷 연결 필요**. (오프라인이면 화면이 검게만 보임)

명령 프롬프트에서:
```cmd
start Playino_NeonRun\index.html
```

## 조작
| 키 | 동작 |
|---|---|
| ← → / A D | 좌우 차선 이동 |
| ↑ / Space / W | 점프 |
| ↓ / S | 슬라이드 |
| (모바일) 스와이프 | 상=점프, 하=슬라이드, 좌우=차선 |

## 장애물
- 🔴 **빨강 벽** → 차선 바꿔 피하기
- 🟠 **주황 허들** → 점프
- 🔵 **파란 가로막** → 슬라이드
- 🟡 **코인** → 먹으면 점수 +

속도는 시간이 지날수록 빨라집니다. 최대 거리 + 코인이 점수.

## 구현 메모
