# 배선도 — 2WD RC카 (L293D 쉴드 기준)

## 핀 점유 한눈에
```
[ L293D 쉴드가 쓰는 핀 — 손대지 말 것 ]
 모터 PWM   : D3(M2) D5(M3) D6(M4) D11(M1)
 시프트레지스터: D4 D7 D8 D12
 서보        : D9 D10

[ 남는(자유) 핀 — 여기에 연결 ]
 D2, D13, A0, A1, A2, A3   (A4/A5는 I2C 예비로 비움)
```

## 연결표
| 부품 핀 | → 아두이노 | 비고 |
|---|---|---|
| **블루투스 TX** | **A0** (SoftwareSerial RX) | 모듈이 보내는 신호를 보드가 받음 |
| **블루투스 RX** | **A1** (SoftwareSerial TX) | ⚠️ 5V→3.3V **분압 필요** (아래) |
| 블루투스 VCC | 5V | 모듈에 따라 3.6~6V |
| 블루투스 GND | GND | |
| **SR04 Trig** | **A2** | |
| **SR04 Echo** | **A3** | |
| SR04 VCC / GND | 5V / GND | |
| 좌 모터 | 쉴드 **M1** 단자 | |
| 우 모터 | 쉴드 **M2** 단자 | |
| 모터 배터리 | 쉴드 **EXT_PWR (+/−)** | 6~9V, **USB 아님** |

> 모터 배터리 **GND는 아두이노 GND와 공통**으로 반드시 연결.

## 레벨 시프트 (아두이노 A1 → 블루투스 RX)
아두이노 출력은 5V, 블루투스 RX는 3.3V 논리입니다. 저항 2개로 낮춥니다.
```
  A1(5V) ──[ 1kΩ ]──┬── BT RX
                     │
                   [ 2kΩ ]
                     │
                    GND
   → 분압 출력 ≈ 5V × 2k/(1k+2k) ≈ 3.3V
```
- 반대 방향(BT TX 3.3V → 아두이노 A0)은 3.3V가 5V 보드에서 HIGH로 인식되어 **직결 OK**.
- 모듈 브레이크아웃에 자체 레벨시프트가 있으면 분압 생략 가능(데이터시트 확인).

## 업로드 팁
블루투스를 **A0/A1(SoftwareSerial)** 에 물렸으므로 **USB 업로드 시 선을 뽑지 않아도** 됩니다.
(D0/D1 하드웨어 시리얼에 물리면 업로드마다 분리해야 함 → 그래서 A0/A1 사용)

## 모듈별 차이 (배선은 동일)
- **HC-06 / HM-10 모두 배선은 위와 동일** (VCC/GND/TX/RX 4선).
- 차이는 **폰 연결 방식**뿐: HC-06=Bluetooth Classic, HM-10=BLE.
  → 연결 방법은 `01_bt_echo/README.md` 참고.
